MERN Stack is a compilation of four different technologies that work together to develop dynamic web apps and websites.

It is a contraction for four different technologies as mentioned below:

  • M - MongoDB
  • E - ExpressJS
  • R - ReactJS
  • N - NodeJS

The MEAN stack is a JavaScript-based framework for developing web applications.

It is a contraction for four different technologies as mentioned below:

  • MongoDB — document database
  • Express(.js) — Node.js web framework
  • Angular(.js) — a client-side JavaScript framework
  • Node(.js) — the premier JavaScript web server

TO BECOME WEB DEVELOPER

WEB DEVELOPMENT is like building a house on the internet. Just as you need architects, builders, and decorators to construct a physical house, you need web developers, designers, and content creators to create a website or web application.

   For this following steps are followed:

  • Web Design
  • Content Creation
  • Frontend Development
  • Backend Development
  • FullStack Development
  • Security
  • Version Control
  • Responsive Design
  • Database Management

FRONTEND WEB DEVELOPMENT is like designing the parts of a car that you see and interact with, such as the steering wheel, dashboard, and buttons. Frontend developers work to make websites and web applications look good and function smoothly for users.

   Frontend web development involves:

  • UI Design
  • HTML
  • CSS
  • JS
  • Responsive Design
  • Cross Browser compatibility
  • Integration
  • Performance Optimization

BACKEND WEB DEVELOPMENT is a crucial aspect of web development that focuses on the server-side of web applications. It involves building and maintaining the behind-the-scenes infrastructure that powers websites and web services.

   Backend development involves:

  • Server Logic
  • Database Management
  • Server Configuration
  • API Development
  • Security
  • Performance Optimization
  • Version Control
  • Server Maintenance
  • Easily Accessible

HTML (HyperText Markup Language) is the standard language used for creating and structuring web pages across the World Wide Web. HTML is not a programming language but rather a markup language that defines the organization and layout of web content.

Within HTML documents, content is arranged hierarchically, enclosed within opening and closing tags. To ensure consistency and compatibility across different web browsers, HTML adheres to standards set by the World Wide Web Consortium (W3C).

CSS (Cascading Style Sheets) CSS is an essential technology in web development that controls the presentation and layout of web pages. It works alongside HTML, the markup language for web content, by defining how that content should be visually displayed in web browsers.

The term "cascading" in CSS refers to the ability to combine styles from different sources, including external stylesheets, internal styles embedded within the HTML document, and inline styles applied directly to individual elements.
